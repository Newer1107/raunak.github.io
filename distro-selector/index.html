About The Project

The Linux Distro Selector is a web application designed to help users find suitable Linux distributions based on their specific needs and hardware. Users provide information about their system, preferences, and use case, and the application leverages the Google Generative AI API to provide two tailored recommendations.

This interactive document provides a comprehensive look into the project's architecture, components, and logic, allowing you to explore how it's built and how it works.
