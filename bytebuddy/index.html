System Guardian Bot

A powerful Discord bot for real-time server monitoring and remote management. Keep an eye on your system resources, execute commands, and check your game server status, all from the comfort of Discord.

Add to Discord

Core Features

Remote Console

Execute shell commands directly and securely on your server from a dedicated Discord channel.

Live System Monitoring

Get real-time updates on CPU, RAM, disk, and network usage with the `!sysinfo` command or a live-updating embed.

Minecraft Server Status

Check the status, player count, and latency of your Minecraft server instantly with the `!ip` command.

Commands at a Glance

A selection of useful commands to get you started.

!ping

Checks the bot's response time.

!uptime

Shows the system and bot uptime.

!sysinfo

Displays a detailed report of system resources.

!top

Shows the top 5 memory-consuming processes.

!ip

Checks the status of the configured Minecraft server.

!restart / !shutdown

Owner-only commands to manage the bot process.
