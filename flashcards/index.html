How It Works

From signing up to acing your quiz, here’s a step-by-step journey through the app.

📝

1. Create & Manage

Sign up and start creating your own flashcards. You can add them one by one or import a whole set from a CSV file.

🧠

2. Take a Quiz

Test your knowledge with our interactive quiz. Questions are shuffled to keep you on your toes.

💬

3. Get AI Feedback

After each answer, our AI, powered by Google's Gemini, provides instant feedback, explaining concepts and helping you learn faster.

API Endpoint: `POST /quiz/answer`

This endpoint takes a user's answer, compares it to the correct one, and uses the Gemini API to generate a helpful explanation.


// Simplified logic from routes/quiz.js
router.post('/answer', async (req, res) => {
  const { userAnswer, correctAnswer, question } = req.body;
  
  const prompt = `
    Question: ${question}
    Correct Answer: ${correctAnswer}
    User's Answer: ${userAnswer}
    Explain if the user is correct or not.
  `;

  // Call to Google Generative AI
  const aiResponse = await getAiFeedback(prompt);

  res.json({ message: aiResponse });
});
